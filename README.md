# Análisis de Métricas de Negocio con SQL y BigQuery

## Descripción
Análisis del dataset real de Olist, el marketplace de e-commerce 
más grande de Brasil, con más de 99,000 pedidos entre 2016 y 2018.

## Objetivo
Identificar oportunidades de mejora en ventas, retención y logística 
usando SQL en BigQuery.

## Herramientas
- BigQuery
- SQL
- Looker Studio
- GitHub

## Insights Clave
- El 97% de los pedidos fueron entregados exitosamente
- Olist creció 27x en volumen entre 2016 y 2017
- Ticket promedio estable en $154 durante todo el crecimiento
- Tarjeta de crédito representa el 74% de las transacciones
- NPS Proxy de +62, considerado excelente en e-commerce
- Solo el 3% de clientes realizó una segunda compra
- São Paulo recibe pedidos 10 días antes de lo prometido
